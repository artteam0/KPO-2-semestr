Расширения исходных файлов проекта С++ могут включать, но не ограничиваются:

.cpp - файлы исходного кода на C++.
.h или .hpp - заголовочные файлы.

Этапы обработки исходного кода программы включают:

Препроцессинг: Обработка директив препроцессора, подстановка макросов, включение содержимого заголовочных файлов.
Компиляция: Преобразование исходного кода на языке программирования в машинный код.
Компоновка: Объединение объектных файлов в исполняемый файл или библиотеку.
Файлы с расширением .obj образуются в результате этапа компиляции. Они содержат объектный код, который еще не связан с другими объектными файлами или библиотеками.

Файл с расширением .exe образуется в результате этапа компоновки (линковки).

После вызова команды cl создаются:

.obj файлы, содержащие скомпилированный код каждого исходного файла.
Если используется предкомпилированный заголовочный файл stdafx.h, то файл с расширением .pch также может быть создан.

В результате выполнения команды link создаются:

Исполняемый файл с расширением .exe.
Файлы DLL или статические библиотеки, если это указано в параметрах.

Прототип функции - это объявление функции, содержащее только ее сигнатуру (имя функции, типы и порядок параметров) и тип возвращаемого значения. Прототипы функций используются для объявления функций до их фактического определения в коде. Это позволяет компилятору знать о существовании функций до их вызова в других частях программы.

Компиляцию в командной строке разработчика можно выполнить с помощью команды cl (Visual C++ компилятор) и указанием соответствующих параметров, таких как исходные файлы, флаги компиляции и т.д.

Компоновку в командной строке разработчика можно выполнить с помощью команды link и указанием объектных файлов (.obj), библиотек и других необходимых параметров.

Для запуска приложения в командной строке разработчика можно использовать команду с указанием пути к исполняемому файлу (.exe). Например, если ваше приложение называется my_program.exe и находится в текущем каталоге, выполните my_program.exe.
